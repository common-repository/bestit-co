This plugin adds a few capabilities to WordPress dashboard for BestIT customers.

== Description ==

this plugin is designd for all the customers of BestIT company

A few features of this plugin:

*   Show last news from bestit website in wordpress dashboard
*   Show last articles from bestit website in wordpress dashboard

== Installation ==

Install bestit-co on a WordPress site

1.Upload the full bestit-co directory into your wp-content/plugins directory.

2.Activate the plugin through the 'Plugins' menu in WordPress.

3.After activating the plugin ,it is automatically added to your wordpress site without any configuration.

== Frequently Asked Questions ==

= Why this widget is not appeared in my dashboard? =

If bestit-co widget is not seen in your dashboard, you can simply go to the "page setting" tab above the dashboard and select the "bestit besides you" to view it.
